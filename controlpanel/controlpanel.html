Farewell, Legacy

This is the last round of HE1 Legacy, enjoy! As you may know, we've been rewriting a new version of HE1 (reusing the brand new HE2 code) on a project named HEBorn.

Let me explain a little what to expect from the new HE1. It's totally different from Legacy, except maybe for the hacking theme (can't change that!).

The new HE1 emulates an operating system (like Windows or MacOS). It has its own applications (File Explorer, Log Viewer, Task Manager, Web Browser etc) on their own windows. You can have multiple windows open at the same time, like the Log Viewer of your target's server and the Task Manager from your own server. It has an icon dock, which allows you to quickly switch between windows, open new windows or de-minimize them.

Some applications can have two contexts: Remote or Gateway. For example, using the Log Viewer you can view your own logs (Gateway context) or the target's logs (Remote context). You can quickly switch between contexts on the same window, without having to open a new one. Each window, as well as the desktop background and the icon dock, have support for different right-click options. Just like you can Right-click > Remove file on your File Browser, you can do so on the new HE1. All of this on your browser, without having to download anything. All of this really fast (60fps). All of this in real time, without having to request a new page on each click. And most of this is ready (the "hard" part is).

Just like HE1 Legacy, HEBorn will be free (and without pay-to-win features). Unlike Legacy, though, the new HE1 will NOT have any sort of advertisement. If all goes well, we plan to keep HE1 a truly free game, sponsored by HE2. If not, we may add premium/pro accounts with cool features like mobile push-notification, custom themes etc to keep the servers running.

Roadmap to the new HE1

  • TODAY (Apr 23rd) we start migrating accounts, as well as a micro-scale closed test (with only a couple of users to test some stuff, real data backup, monitoring etc).
  • Wednesday (Apr 26th) we'll send the first batch of invitations to the closed test. I'll publish the invitee list soon.
  • Next Monday (May 1st) we'll send the second batch of invitations to the closed test. The list will be made public too.
  • Next next Monday (May 7th) we plan to open HEBorn for public registration.
  • On May 29th we will shut down HE1 legacy, reset all HEBorn accounts and start the very first round of the new HE1.

Account Migration

The only thing we'll migrate from HE1 Legacy to the new HE1 is your username. All rest will be lost forevah[1].

When signing up for HEBorn, if you pick an username already taken on HE1 Legacy, you'll be asked to login on HE1 Legacy using that username, click on a "Migrate" button that will take you to a special page, and then proceed with the account creation. You don't need to have access to your HE1 Legacy email address to complete the migration.
This way, you have the guarantee that your current username will be yours on the new HE1, if you choose to.

To avoid inactive accounts from taking up usernames needlessly, this migration process will be on-line for 3 months. After that period (Jul 23rd), the migration check will be disabled and anyone can register any unclaimed HE1 Legacy username.

Farewell Message

Once HE1 Legacy is shut down, the legacy.hackerexperience.com address will keep a nostalgic compilation of HE1 Legacy resources, including the read-only pages and some screenshots. We also want to add a few, selected farewell messages from our players.

If you have any sentiment for HE1 Legacy, if you made great friends (or enemies!), or if you spent uncountable days of your life here, consider writing a farewell message. If selected, it will be forever engraved on our Legacy Nostalgy webpage. Leave your message here.
[1] - We plan to keep read-only pages of the hall of fame, (old) forum, profiles and game stats. If you want to keep something special (even if it's only for you), please let us know.
